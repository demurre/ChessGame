# Chess Play&Learn

## Chess Play&Learn with 3 pages (Main, Play and Learn). The Main page contains randomly selected cards with links to learning and playing. The Play page contains a chessboard with a list of moves and a take back button. The Learn page contains chess terms and opening cards with a carousel.
